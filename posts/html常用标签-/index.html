HTML 常用标签

重点内容为 4 部分：a 标签、table 标签、img 标签和 form 标签。

1. a 标签的用法 href

  • 跳转到路径或者外部网址，href：网址、路径、伪协议（宇宙唯一解 javascript:;)、id。
  • 发邮件 e.g href=“mailto:xxx@cc”
  • 打电话 e.g href=“tel:137xxxxxx”
  • 另外提到 iframe，听懂了，但是呢作为小白，默默走开就好，后面再说。

2. img 标签的用法

发出 get 请求，展示一张图片

4 个属性

  • alt：加载失败提示
  • width：设置图片宽度，100%为自适应屏幕大小，手机来说高度不设。
  • height：设置图片高度。建议宽和高二选一，不要搞得变形恶心自己。
  • src：图片路径，显示的图片在哪里。

2 个事件

  • onload：加载成功
  • onerror：加载失败 搭配 javascript 可以做深层次的统计，小白飘过就好~

3. table 标签的用法

就是表格，默认的格式很丑，需要重置格式。下面 2 行重置格式的万金油，记住了。

border-collapse：collapse；
border-spacing：0；

3 个元素

  • thead：排头
  • tbody：中间，一般有值部分
  • tfoot：最下面，总和 统计之类

单元格组成 记住 3 个

  • th 头
  • tr table row
  • td 值

4. form 标签的用法

注意事项 3 点

  • 一般不监听
  • 需要有 name
  • 需要 submit
