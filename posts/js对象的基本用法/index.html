JS基本语法简介

October 17, 2019

JS有7中数据类型，其中对象是JS中一种相对复杂的数据类型。

声明对象的两种语法

第一种：

let obj = { 'name': 'frank', 'age': 18 }

第二种

let obj = new Object({'name': 'frank'})

记住：不要使用var。 重要的事情说三遍：键名是字符串，不是标识符，可以包含任意字符，即使省略引号也是字符串。

如何删除对象的属性

删除对象的属性按照用下面代码：

delete obj.xxx 或者delete obj['xxx']

检查是否已经删除

'xxx' in obj ===false

如何查看对象的属性（读属性）

  1. 查看自身所有属性

    Object.keys(obj)

  2. 查看自身+共有属性

    console.dir(obj)

    或者自己依次用Object.keys打印出obj.proto

  3. 判断一个属性是共有属性还是自己的属性

    obj.hasOwnProperty('toString')

如何修改或增加对象的属性(写属性)

  1. 直接赋值

    let obj = {name: 'frank'} // name是字符串
obj.name = 'frank' // name是字符串
obj['name'] = 'frank'
~~obj[name] = 'frank'~~ // name是变量，值不确定，错
obj['na'+'me'] = 'frank'
let key = 'name'; obj[key] = 'frank'
let key = 'name'; ~~obj.key = 'frank'~~//错，因为obj.key等价于obj['key']

  2. 批量赋值

    Object.assign(obj, {age: 18, gender: 'man'})

注意，一般无法通过自身修改或增加共有属性，只有通过修改或增加原型上的属性。如果要修改，就在一开始就改，后面中途改会导致很多问题，推荐使用Object.create。

let obj=Object.create(common)

所有的proto代码都是强烈不推荐写的。

最后，说明一下 ‘name’ in obj和obj.hasOwnProperty(‘name’) 的区别。

  1. ‘name’ in obj//检查是否含有这个属性，不区分共有和自身属性。
  2. obj.hasOwnProperty(‘name’) //检查是否为自身的属性。
