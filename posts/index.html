HTML常用标签

HTML 常用标签 重点内容为 4 部分：a 标签、table 标签、img 标签和 form 标签。 1. a 标签的用法 href 跳转到路径或者外部网址，href：网址、路径、伪协议（宇宙唯一解 javascript:;)、id。 发邮件 e.g href=“mailto:xxx@cc” 打电话 e.g href=“tel:137xxxxxx” 2. img 标签的用法 发出 get 请求，展示一张图片 加入笔记看看吧 3. table 标签的用法 就是表格，默认的格式很丑，需要重置格式。下面 2 行重置格式的万金油，记住了。 border-collapse：collapse； border-spacing：0； 其他的看笔记 注意 3 点 一般不监听 需要有 name 需要 submit 其他看笔记。来张照片看笔记 感想 初次接触到 HTML，以加倍 1.2~1.5 的去听讲，听完了一顿懵，老老实实跟着老师 key 一边，还是很有感觉的。 学走路的时候最难，跑起来就容易了，慢慢磨吧。

HTML入门笔记

大家好 我的博客开通了，希望我能写出好文章给大家看 1。 HTML 入门笔记 HTML 是谁发明的？ 回答：李爵士（Tim Berners-Lee） HTML 起手应该写什么 回答：！+tab 键 常用的表章节的标签有哪些，分别是什么意思（h1~h6、section、article、main、aside 等等） 回答： h1~h6 标题 section 章节 article 文章 main 主要内容 aside 旁支内容 p 段落 header 头部 footer 脚部 div 划分 全局属性有哪些 class 类 contenteditable 网页内容可编辑 hidden 隐藏 id 唯一标签，但不报冲突 style 样式 tabindex tab 键的响应顺序 title 鼠标悬浮显示的内容 常用的内容标签有哪些，分别是什么意思（a、strong、em、code、pre 等等） ol + li 有序列表 ul + li 无序列表 dl + dt +dd 描述型列表 pre 保留空格 hr 水平分割线 br 换行 a 链接 em 强调形式 斜体 strong 强调内容 粗体 code 代码 quote 短文本引用 blockquote 一段引语
