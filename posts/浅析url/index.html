浅析URL

October 6, 2019

URL(Uniform Resource Locator)为统一资源定位符，是蒂姆·贝纳斯·李（TimBemers—Lee)提出的 WWW 其中一部分，表示如下：

WWW=RUL+HTTP+HTML

URL 由 6 部分组成

URL=协议+域名或 IP+端口号+路径+查询字符串+锚点

协议

规定请求和响应的格式的文件，如 http 或者 https。

域名

域名就是 IP 的别称，一个域名可以对应不同的 IP，通过 DNS(Domain Name System)域名系统进行对应。 如 www.baidu.com 的 2 个 IP 分别是 39.156.69.79 和 220.181.38.148。

  • DNS 的作用

简单的说就是一个将域名翻译成 ip 地址的系统。它作为将域名和 IP 地址相互映射的一个分布式数据库，能够使人更方便地访问互联网。

  • IP 的作用

就是定位一台设备，按照路由器（网关）隔开分为内网和外网，一个设备有自己特有的内网 IP 和外网 IP，相互不干扰，内网与外网连接必须通过路由器。如家里的手机、电脑、电视各自拥有的 IP 叫内网 IP,然而上网都要通过路由器，外网 IP 是同一个。但是每次重启路由器后，运营商会重新分配外网 IP。

  1. ping 命令看自己访问的 IP

      ping baidu.com

  2. nslookup 命令查看所有 IP

      nslookup baidu.com

  3. IP138.com 查看自己的外网 IP

端口号

TCP（Transmission Control Protocol）端口就是为 TCP 协议通信提供服务的端口。

  1. 80 对应 HTTP 超文本服务器
  2. 443 对应 HTTPS 安全服务
  3. 21 对应 FTP 文件传输

路径

请求的文件在服务器的路径，在哪里。

查询字符串

找的内容的关键字。

锚点

找的内容的某一部分。

域名的分类

顶级域名

国家域名可分为不同级别，包括顶级域名、二级域名，三级域名等。

顶级域名又分为两类：

一是国家顶级域名（national top-level domainnames，简称 nTLDs），有 200 多个国家都按照 ISO3166 国家代码分配了顶级域名，例如中国是 cn，美国是 us。

二是国际顶级域名（international top-level domain names，简称 iTDs），例如表示工商企业的 .com .top，表示网络提供商的.net，表示非盈利组织的.org 等。大多数域名争议都发生在 com 的顶级域名下，因为多数公司上网的目的都是为了赢利。

二级域名

二级域名是指顶级域名之下的域名，在国际顶级域名下，它是指域名注册人的网上名称，例如 ibm，yahoo，microsoft 等；在国家顶级域名下，它是表示注册企业类别的符号，例如 com，top，edu，gov，net 等。

三级域名

三级域名用字母（ A ～ Z，a ～ z，大小写等）、数字（0 ～ 9）和连接符（－）组成， 各级域名之间用实点（.）连接，三级域名的长度不能超过 20 个字符。 如无特殊原因，建议采用申请人的英文名（或者缩写）或者汉语拼音名 （或者缩写） 作为三级域名，以保持域名的清晰性和简洁性。
