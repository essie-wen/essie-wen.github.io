JS函数的执行时机

在使用JS函数一定要考虑执行时机，比如如下一段代码，本意是要打印出0~5，可结果却是6个6。怎么理解呢？一起看看吧。 let i = 0 for(i = 0; i<6; i++){ setTimeout(()=>{ console.log(i) },0) } // 结果为6，6，6，6，6，6 首先，setTimeout的设定为0，表示立即执行，前提是要完成当前的循环。 当i=0，标记要执行1次打印i（执行时机setTimeout在完成当前循环后）； 当i=1，标记要执行1次打印i（执行时机setTimeout在完成当前循环后）； 当i=2，标记要执行1次打印i（执行时机setTimeout在完成当前循环后）； 当i=3，标记要执行1次打印i（执行时机setTimeout在完成当前循环后）； 当i=4，标记要执行1次打印i（执行时机setTimeout在完成当前循环后）； 当i=5，标记要执行1次打印i（执行时机setTimeout在完成当前循环后）； 当i=6，结束循环，可以开始执行打印i（总共标记了6次，所有要打印6次，此时i=6了呀） 所有 得到了6个6. 怎么打印出0~5呢，ES6做了优化。超级简单，把let放入for循环即可，万能的let&for，没有为什么，JS之父made it，记住就好。 for(let i = 0; i<6; i++){ setTimeout(()=>{ console.log(i) },0) } // 结果为6，6，6，6，6，6 再简单一点，直接拿掉setTimeout也可以。打印出0~5. let i = 0 for(i = 0; i<6; i++){ console.log(i) }
